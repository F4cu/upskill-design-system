# Colors

Each hue has 12 steps. Steps 1–4 are tinted backgrounds, 5–8 are borders and dividers, and 9–12 are solid fills and high-contrast text.

## Full palette

Never mix scales on the same token. Use the light scale in light theme, the dark scale in dark theme.

## Terracotta

The brand hue. Use steps 8–10 for interactive elements and CTAs, steps 1–3 for tinted backgrounds.
