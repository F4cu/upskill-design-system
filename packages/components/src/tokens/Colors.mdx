# Colors

Each hue has three sub-scales: a light scale (1–12, backgrounds → text), a dark-mode scale, and an alpha (transparent) variant. Use the light scale in light theme, the dark scale in dark theme. Never mix scales on the same token.

## Terracotta

The brand hue. Use steps 8–10 for interactive elements and CTAs, steps 1–3 for tinted backgrounds.
