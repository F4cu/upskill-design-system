# Typography

The type scale has 11 steps from `fontSize1` (0.75rem / 12px) to `fontSize11` (2rem / 32px), all in rem so they scale with the user's browser font size preference. Line-heights are unitless ratios — never fixed px values.

## Line Heights

Line heights are unitless ratios. Use the lower values for large display text (tighter optical fit), higher values for body copy (readability).

| Value | Typical use |
|---|---|
| `1` | Display / hero headings |
| `1.25` | Large headings |
| `1.4` | Subheadings, UI labels |
| `1.5` | Body copy (default) |
| `1.75` | Small print, captions |
