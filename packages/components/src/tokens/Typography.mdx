# Typography

The type scale has 11 steps from `font.size.1` (0.75rem / 12px) to `font.size.11` (2rem / 32px), all in rem so they scale with the user's browser font size preference. Line-heights are unitless ratios — never fixed px values.

## Font Family

Two typefaces cover all roles in the system. **Playfair Display** is the headline and display face — a high-contrast serif drawn for large sizes. **Roboto** handles body copy, UI chrome, and documentation.
