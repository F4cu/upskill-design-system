# Border Radius

Five steps from sharp (`null`) to fully rounded (`lg`). Click any swatch to copy the token name.

<BorderRadiusGrid />

## Usage guidance

| Token | Use case |
|---|---|
| `borderRadiusNull` | Data tables, code blocks, strictly geometric UI |
| `borderRadiusXs` | Tags, badges, small chips |
| `borderRadiusSm` | Inputs, select menus, tooltips |
| `borderRadiusMd` | Cards, modals, panels |
| `borderRadiusLg` | Buttons, large interactive surfaces |
