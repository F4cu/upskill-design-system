# Border Radius

Five steps from sharp to fully rounded. Click any swatch to copy the CSS variable.

<BorderRadiusGrid />

## Usage guidance

| Token | Use case |
|---|---|
| `--ds-border-radius-null` | Data tables, code blocks, strictly geometric UI |
| `--ds-border-radius-xs` | Tags, badges, small chips |
| `--ds-border-radius-sm` | Inputs, select menus, tooltips |
| `--ds-border-radius-md` | Cards, modals, panels |
| `--ds-border-radius-lg` | Buttons, large interactive surfaces |
